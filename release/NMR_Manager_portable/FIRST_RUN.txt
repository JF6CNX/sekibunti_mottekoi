NMR Manager 初回起動ガイド

1. zipファイルを好きな場所に展開します。
2. "NMR Manager.exe" をダブルクリックします。
3. Windows の警告が出た場合は、「詳細情報」から実行します。
4. アプリ内の "Settings" を開きます。
5. "NMR input folder" に、NMR のサンプルフォルダが入っている場所を指定します。
6. "Excel output folder" を確認します。Excel の保存先を変えたい場合はここを変更します。
7. 必要ならテーマを選びます。
8. "Save" を押します。
9. メイン画面に戻り、サンプル一覧が表示されることを確認します。
10. サンプルを選んで Excel 出力します。

補足
- "data" フォルダと "output" フォルダは EXE と同じ場所に置いたまま使ってください。
- サンプルが表示されない場合は、"NMR input folder" の指定先を確認してください。
- Excel ファイルは、設定を変えない限り "output" フォルダに保存されます。
