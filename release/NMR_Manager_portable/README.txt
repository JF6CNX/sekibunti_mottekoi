NMR Manager portable package

How to use
1. Extract this folder anywhere on your PC.
2. Run "NMR Manager.exe".
3. Open Settings in the app.
4. Set "NMR input folder" to your own NMR data folder.
5. Set "Excel output folder" if you want a different export location.
6. Save settings and use the app.

Notes
- Keep the "data" and "output" folders next to the EXE.
- Windows may show a SmartScreen warning because this app is not code-signed.
- If that happens, click "More info" and then run the app.
